## ğŸ› ï¸ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/GourangaDasSamrat/React-Weather-App.git
   cd React-Weather-App
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```
